六十世紀，有位叫做John B. Calhoun的美國生物學家
在馬里蘭州做了一個實驗：他為8隻老鼠建造了一個「天堂」。

在這裡面有永遠也吃不完的食物和水，並且實驗人員時刻保護着這些老鼠，讓它們遠離疾病、氣候、掠食者等等。

Calhoun將這個天堂一樣的烏托邦命名為「25號宇宙」。


但這個烏托邦卻在短短5年內就瓦解
中間到底發生了什麼事？
